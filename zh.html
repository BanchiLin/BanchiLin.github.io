代表项目

JobMate：基于同伴经验的 AI 职业探索系统

2025.10 – 2026.04

JobMate 是一个将小红书上的真实求职经验分享内容转化为可对话 AI 角色的交互系统， 旨在帮助年轻求职者从被动浏览转向主动的职业探索与意义建构。

角色：项目负责人；系统设计；数据收集与处理；提示词设计； 用户研究；定性分析；论文写作。

方法：LLM 数据处理、角色卡构建、自我决定论、原型设计、用户研究。

状态：论文、代码、视频与网页原型目前未公开。

融合情绪与 Emoji 表征的股价预测研究

2025.07 – 2025.09

该项目基于 StockEmotions 数据集，探究 emoji 情绪编码是否能够提升股价预测效果。

角色：文献调研；结果分析；论文撰写。

方法：LSTM、情绪表征、emoji 编码。

结果：相关论文已发表，代码未公开。

基于 LLM 与 ReAct 的 Text-to-Cypher

2025.03 – 2025.06

该项目探索如何将自然语言问题转化为图数据库查询语言 Cypher。

角色：文献调研；现有工作梳理；基于 LLM 与 ReAct 框架进行实验测试。

方法：大语言模型、ReAct 框架、语义解析。

发现：初步效果有限，准确率约为 20%。

权力概念的隐喻表征心理学实验复现

2022.12 – 2023.02

该心理学实验探究权力概念与字体大小、颜色之间的隐喻联系。

角色：实验程序设计；参与部分实验；完成全部数据分析；撰写部分报告。

方法：Stroop 范式、反应时分析。

结果：当高权力概念以大号或金色字体呈现、低权力概念以小号或灰色字体呈现时， 被试反应时更短。
